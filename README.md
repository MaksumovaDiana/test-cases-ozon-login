🛠 Что сделано

Тест-кейсы (10 шт.):
1. Успешный вход с корректными данными
2. Вход с неправильным паролем
3. Попытка входа с пустыми полями
4. Вход с паролем минимальной длины (6 символов)
5. Ввод email в неправильном формате
6. Переход по ссылке "Забыли пароль?"
7. Переход к регистрации
8. Проверка кнопки "Показать пароль"
9. Ввод пароля с включённым Caps Lock
10. Ввод очень длинного email (более 100 символов)

Чек-лист (24 проверки):
- Функциональные (5): вход, пароль, пустые поля, восстановление, регистрация
- UI/UX (6): кнопка, текст, выравнивание, иконка, мобильная версия, планшет
- Граничные значения (4): длина пароля и email
- Безопасность (3): скрытие пароля, HTTPS, SQL-инъекции
- Нагрузочное (1): 1000 пользователей одновременно
- Совместимость (5): Chrome, Firefox, Safari, iOS, Android
